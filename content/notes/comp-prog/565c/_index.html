Solution
For this problem we can create a vector, c, to keep track of the statuses of our sequences. For example c[0] is how many elements our first sequence has. If c[0] = 1 then our first sequence is 4. If c[0]=4 then our first sequence has 4,8,15, and 16 in that order. After this we need an array, transf to keep track of which sequence we’ll be inserting each element into. Finally we should make sure that when we only insert elements into that sequence if it will preserve the correct order. This is the purpose of check. With these three arrays we iterate through the input and either add elements to the appropriate sequence or increment our output variable. Each time we insert an element, we have to increment transf[element] to make sure it will always have the correct value. Afterwards we iterate through c to add all the indexes of c that don’t equal 6 to our output.
Code

#include <bits/stdc++.h>
typedef int OwO; typedef double UwU;
#define int long long
#define double long double

using namespace std;

OwO main(){
  ios::sync_with_stdio(false);
  cin.tie(NULL);
  int n;
  cin >> n;
  vector<int> c(n);
  int check[43];
  check[4]=0;
  check[8]=1;
  check[15]=2;
  check[16]=3;
  check[23]=4;
  check[42]=5;
  int transf[43];
  transf[4]=0;
  transf[8]=0;
  transf[15]=0;
  transf[16]=0;
  transf[23]=0;
  transf[42]=0;
  int oup = 0;
  
  for(int x = 0; x<n; x++){
    int t;
    cin >> t;
    int idx = transf[t];
    if(c[idx]!=check[t])
      oup++;
    else{
      c[idx]++;
      transf[t]++;
    }
  }

  int idx = 0;
  while(idx < n && c[idx]!=0){
    if(c[idx]<6)
      oup += c[idx];
    idx++;
  }
    
  cout << oup << endl;


  return 0;
}
