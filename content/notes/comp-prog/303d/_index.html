Solution
We can take a greedy approach to this problem. We sort the people based on how long it takes to serve them. Then we iterate through the people. If the current person is satisfied then we increment our output and add their time to the current time waited. If the current person is not satisfied then we skip forward until we find someone who will be satisfied with the current time waited, increment our output and add that personâ€™s time to our time waited. We then start iterating forward from that person we skipped to. Every person between the previously satisfied person and the person we skipped to will be dissatisfied no mater what.

  int oup = 0;
  for(int x = 0; x<t.size(); x++){
    if(tme<=t[x])
      oup++;
    else{
      while(x<t.size() && tme>t[x])
        x++;
      if(x<t.size() && tme<=t[x])
        oup++;
    }
    if(x<t.size())
      tme+=t[x];
  }
  cout << oup << endl;
}
